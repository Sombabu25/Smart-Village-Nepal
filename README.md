# 🌐 Smart Village Nepal

An offline-first civic issue reporting platform designed to empower rural communities by enabling citizens to report local infrastructure problems and track their resolution in real-time.

---

## 🚀 Overview

Smart Village Nepal is a web-based platform that helps bridge the gap between citizens and local authorities.

In many rural areas, issues like road damage, water shortages, and electricity problems often go unreported or take a long time to resolve. This platform provides a simple and accessible way for people to raise concerns and for authorities to act efficiently.

---

## ✨ Features

### 👤 Citizen Side
- 📝 Multi-step issue reporting (category, description, severity)
- 📍 Location capture (GPS + manual fallback)
- 📸 Image upload (in progress)
- 📊 Track issue status (Pending → In Progress → Resolved)
- 👍 Support/upvote system for prioritization
- 🌐 Bilingual interface (Nepali + English)
- 📡 Offline-first support (reports saved locally & synced later)

---

### 🏛️ Admin Dashboard
- 📋 View and manage all reported issues
- 🔍 Filter by category, status, and location
- 📊 Analytics (issue distribution, status counts)
- ⚙️ Update issue status and track progress

---

## 🧠 How It Works

1. User reports an issue with details and location  
2. Data is saved locally if offline  
3. Syncs automatically when internet is available  
4. Admin reviews and updates issue status  
5. Users can track progress in real-time  

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Storage:** Cloudinary / Firebase (for images)  
- **Other:** PWA (Offline-first), Geolocation API
